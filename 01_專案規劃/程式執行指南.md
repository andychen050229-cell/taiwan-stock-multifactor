# 股市趨勢預測導向決策輔助系統 — 程式執行指南

## 你需要執行的程式（按順序）

### Step 1：抓取 OHLCV 資料（約 3.5 小時）
```bash
cd 大數據與商業分析專案
pip install FinMind python-dotenv
# 先建立 .env（填入你的 FinMind token）
cp .env.template .env
# 然後執行
python -m src.data.ohlcv_fetcher
```
- 來源檔案：`src/data/ohlcv_fetcher.py`
- 輸出：`選用資料集/parquet/stock_prices_ohlcv.parquet`
- 支援斷點續傳，中斷後重新執行即可繼續

### Step 2：執行 Phase 1 完整流程
```bash
python run_phase1.py
```
- 來源檔案：`run_phase1.py`（主控台）
- 輸出：
  - `outputs/feature_store.parquet`（特徵商店）
  - `outputs/income_stmt_processed.parquet`（處理後損益表）
  - `outputs/reports/Phase1_品質報告_最新.json`（品質報告）

## 程式碼架構說明（src/）

```
src/
├── config/                    # 設定檔
│   ├── base.yaml             # 基礎設定（所有參數）
│   └── dev.yaml              # 開發環境覆蓋設定
│
├── data/                      # 資料處理模組
│   ├── loader.py             # Parquet 資料載入器
│   ├── ohlcv_fetcher.py      # FinMind OHLCV 抓取（含流量控制+斷點續傳）
│   ├── price_processor.py    # 股價前處理（缺值+除權息+漲跌停）
│   ├── financial_processor.py # 損益表處理（累計→單季+PIT日期+財務比率）
│   ├── label_generator.py    # 標籤生成（D+1/D+5/D+20 三元分類）
│   ├── text_processor.py     # 文本前處理（分類+過濾+去重+清洗）
│   └── leakage_detector.py   # 前瞻偏差偵測（4層檢查）
│
├── features/                  # 特徵工程模組
│   └── engineer.py           # 五大支柱特徵工程
│
├── models/                    # [Phase 2] 模型訓練
├── backtest/                  # [Phase 2] 回測模擬
├── governance/                # [Phase 3] 模型治理
├── ui/                        # [Phase 3] 前端介面
│
└── utils/                     # 通用工具
    ├── config_loader.py      # YAML 設定檔載入器
    ├── logger.py             # loguru 三層日誌
    └── helpers.py            # retry/timer 裝飾器
```
