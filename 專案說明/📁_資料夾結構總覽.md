## 五、快速啟動

### 看課堂錄影簡報
- 雙擊 `台股多因子預測系統_v11.5.18_課堂簡報.pptx`

### 讀逐字稿（錄影用）
- 開啟 `台股多因子預測系統_v11.5.18_逐字稿.md`

### 讀研究報告
- 雙擊 `台股多因子預測系統_v11.5.17_研究報告.docx`

### 啟動網頁儀表板
```bash
streamlit run streamlit_app.py
```

### 重建課堂簡報
```bash
cd 簡報設計
python build_full.py                # 重建 deck_full.pptx
cp deck_full.pptx ../台股多因子預測系統_v11.5.18_課堂簡報.pptx
```

### 重驗治理關卡 9（pipeline）
```bash
cd 程式碼
python verify_pipeline_gate.py      # 應回 6/6 PASS · prediction_pipeline_valid = True
```
