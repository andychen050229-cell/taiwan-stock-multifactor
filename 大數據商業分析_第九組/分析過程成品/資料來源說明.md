## 一、資料分為兩塊

### A · 課程指定資料庫（4 張主表）

教師方提供的核心研究資料庫，作為樣本主鍵與標籤生成依據：

| 資料表 | 筆數 | 用途 |
|---|---|---|
| `companies` | 1,932 | 公司基本識別（JOIN key） |
| `stock_prices` | 877,699 | 標籤生成、val 估值（僅含 closing_price） |
| `income_stmt` | 14,968 | 基本面（fund_）特徵、僅 6 個損益科目 |
| `stock_text` | 1,125,134 | 文本特徵（event_ / txt_ / sent_）來源：PTT / Dcard / Mobile01 / Yahoo |

> 課程指定資料庫不在本壓縮檔內、由教師端提供。

---

### B · FinMind 公開 API 補充資料（6 張表）

針對核心資料庫缺項補齊，使用 [FinMind](https://finmindtrade.com/) 公開 API 抓取：

| 檔案 | FinMind Dataset | 補充內容 |
|---|---|---|
| `stock_prices_ohlcv.parquet` | `TaiwanStockPrice` | 完整 OHLCV（核心表僅含收盤價） |
| `balance_sheet.parquet` | `TaiwanStockBalanceSheet` | 資產負債表（核心庫無此項） |
| `cashflow.parquet` | `TaiwanStockCashFlowsStatement` | 現金流量表（核心庫無此項） |
| `industry.parquet` | `TaiwanStockInfo` | 產業分類欄位（核心庫無此項） |
| `institutional_investors.parquet` | `TaiwanStockInstitutionalInvestorsBuySell` | 三大法人買賣超 |
| `margin_trading.parquet` | `TaiwanStockMarginPurchaseShortSale` | 融資融券交易資料（已於 2026-04-19 因覆蓋率不足下架） |

涵蓋期間：2023-03 ~ 2025-03（505 個交易日 × 1,930 檔股票）。

## 二、抓取程式碼位置

抓取腳本已隨原始碼提交、放在 **分析 / 資料抓取 /** 目錄下：

```
分析/資料抓取/
├── 抓取股價資料.py            FinMind TaiwanStockPrice
├── 抓取資產負債表.py          FinMind TaiwanStockBalanceSheet
├── 抓取現金流量表.py          FinMind TaiwanStockCashFlowsStatement
├── 抓取產業分類.py            FinMind TaiwanStockInfo
├── 抓取法人買賣超.py          FinMind TaiwanStockInstitutionalInvestorsBuySell
├── 抓取融資融券.py            FinMind TaiwanStockMarginPurchaseShortSale
└── 產生推薦組合.py            從預測結果生成 Top 1% 投組
```

執行方式（依序執行即可重建外部資料）：

```bash
cd 分析/資料抓取
python 抓取股價資料.py        # ~30 分鐘（1,930 檔 × 505 日）
python 抓取資產負債表.py
python 抓取現金流量表.py
python 抓取產業分類.py
python 抓取法人買賣超.py
python 抓取融資融券.py        # 已下架、僅作完整性保留
```

## 四、若要完整重現分析

1. 取得 FinMind API key（免費註冊：[finmindtrade.com](https://finmindtrade.com/)）
2. 在環境變數設 `FINMIND_TOKEN=<your_token>`
3. 執行 `分析/資料抓取/` 中各腳本下載 6 張補充資料
4. 將 4 張核心表 + 6 張 FinMind 表放在 `data/raw/` 下
5. 依序執行 `分析/執行Phase1_資料工程.py` ～ `執行Phase6_*.py`
6. 即可重現 v11.5.18 全部結果（OOS AUC 0.6455、9/9 治理關卡通過、24m CAGR +36.8% 等）

完整 6-phase pipeline 詳見 **分析/** 目錄下各 `執行PhaseN_*.py` 檔案。
