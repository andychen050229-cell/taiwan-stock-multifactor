# 技術棧與系統架構說明

## 推薦技術棧

### 後端 / 資料處理
| 類別 | 工具 | 用途 |
|------|------|------|
| 程式語言 | Python 3.10+ | 主要開發語言 |
| 資料處理 | pandas, polars | 結構化資料處理 |
| 機器學習 | scikit-learn, XGBoost, LightGBM | 預測模型 |
| 深度學習 | PyTorch / TensorFlow | 文本模型/序列模型 |
| NLP | transformers (HuggingFace), jieba | 中英文文本處理 |
| 回測引擎 | backtrader / zipline-reloaded | 策略回測 |
| 可解釋性 | SHAP, LIME | 模型解釋 |
| API 框架 | FastAPI | 後端 API 服務 |
| 排程 | APScheduler / cron | 定時更新 |
| 資料庫 | PostgreSQL / SQLite | 資料儲存 |

### 前端
| 類別 | 工具 | 用途 |
|------|------|------|
| 框架 | React / Next.js | 前端 SPA |
| 視覺化 | Recharts / D3.js / Plotly | 互動圖表 |
| UI 元件 | Tailwind CSS / shadcn/ui | 介面設計 |
| 狀態管理 | Zustand | 前端狀態 |

### 資料來源 API
| 來源 | 市場 | 免費額度 | 資料類型 |
|------|------|----------|----------|
| Yahoo Finance (yfinance) | 美股/台股 | 無限制 | OHLCV, 財報 |
| FinMind | 台股 | 每日 500 次 | 完整台股資料 |
| Alpha Vantage | 美股 | 25 次/日 | OHLCV, 基本面 |
| FRED | 全球 | 無限制 | 總經指標 |
| NewsAPI | 全球 | 100 次/日 | 新聞文本 |
| TWSE OpenAPI | 台股 | 無限制 | 即時成交 |

### 部署方案
| 方案 | 優點 | 缺點 | 適合情境 |
|------|------|------|----------|
| Vercel + Supabase | 免費、快速部署 | 計算資源有限 | MVP 展示 |
| GitHub Pages + Cloud Run | 免費靜態 + 彈性後端 | 設定較複雜 | 學期專案 |
| Streamlit Cloud | 零配置、快速 | 自訂受限 | 快速原型 |
| Render | 免費方案、自動部署 | 冷啟動慢 | 中型專案 |

## 系統架構圖（文字描述）

```
[使用者] → [前端 React/Next.js]
              ↕
         [FastAPI 後端]
         ↙    ↓    ↘
[資料Pipeline] [模型服務] [報告生成]
    ↓            ↓          ↓
[PostgreSQL]  [預測引擎]  [決策轉譯]
    ↑            ↑
[排程更新]   [特徵倉庫]
    ↑
[外部API: yfinance/FinMind/NewsAPI]
```
